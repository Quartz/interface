# ArticleStrip

A summary of a Quartz article, typically used to link to that article.

## Usage guidelines

- **Do** update the `ArticleStrip` component if you make improvements that other components could benefit from.
- **Do not** use the `ArticleStrip` component.

## Props

<Canvas>
	<Story
		args={{
			kicker: 'Lessons from language',
