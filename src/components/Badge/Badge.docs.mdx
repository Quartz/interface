# Badge

A `Badge` is a round image of predetermined size.

## Usage

```js
import { Badge } from '@quartz/interface/src/components';
```

## Props

<Props of={Badge} />

## Usage guidelines

- **Do not** attempt to render a `Badge` at a size other than the built-in sizes.
