# Badge

A `Badge` is a round image of predetermined size used for decorative images.

## Usage

```js
import { Badge } from '@quartz/interface/src/components';
```

## Props

<Props of={Badge} />

## Usage guidelines

- **Do** use `Badge` for avatars and other representational images (e.g., an image representing a series or field guide).
- **Do not** use `Badge` for editorial images.
- **Do not** attempt to render a `Badge` at a size other than the built-in sizes.
