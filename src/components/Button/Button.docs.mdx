# Button

Presents the user with an action to take, such as a form submission or promotional call-to-action.

## Usage

```js
import { Button } from '@quartz/interface/src/components';
```

```js
import { ButtonLink } from '@quartz/interface/src/components';
```

## Props

<Props of={Button} />

## Usage guidelines

- **Do** use Button for actions that change something on the page, such as form submissions or toggling content visibility.
- **Do not** use Button for linking to another page, use `ButtonLink` instead.
