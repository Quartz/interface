# EmojiList

A list of items with a descriptive emoji as a bullet point. There are two components exported for use:

- Use `EmojiList` when pulling in a bulleted list from the post content of a WordPress post. `EmojiList` parses raw html, pulls out the emojis, and styles the list appropriately.

- Use `StructuredEmojiList` when passing an array of objects into EmojiList‚Äîe.g., when you are creating an emoji list from code.
