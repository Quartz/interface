# Fieldset

A basic uncontrolled input component that for either a `textArea` or `input` element depending on the `isMultiline` prop.
Rendered inside an associated `label` tag. The `inputRef` prop can be used to find the component's `value`.
Generally, this is not meant to be used by itself, but as a building block within other components.

## Props

export const defaultActions = {
