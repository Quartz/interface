# PageHeader

Very large text that can be used to present the title and short additional intro text for a non-article page.

It takes optional children that should be used for relevant non-text components, such as navigation elements.

## Props

<Props of={PageHeader} />

## Usage Guidelines

- **Do not** use this on a page where an H1 already exists
- **Do not** use the optional children for general page content

## Examples

### Default

<Canvas>
