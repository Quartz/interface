# TextInput

A basic input component for either single or multiline input. Use component props to add a label above
the element, set whether or not it is a required field, add validations, set input type for use with
browser validations (such as `email` or `password`), and pass functions that will be used when a user
interacts with the component.

## Props

export const defaultActions = {
	onBlur: action( 'blurred' ),
	onChange: action( 'changed' ),
	onFocus: action( 'focused' ),
